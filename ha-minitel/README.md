# Minitel for Home Assistant

Browse and control your Home Assistant instance from a Minitel terminal or emulator.

![Minitel](icon.png)

## About

This Home Assistant add-on turns your Minitel terminal (or emulator) into a home automation controller. It exposes a WebSocket server on port 3615 and optionally connects to a physical Minitel via serial port.

## Features

- Navigate rooms and entities using classic Minitel function keys
- Toggle lights, switches, and other entities
- Control brightness, temperature, and cover positions
- View and trigger automations
- Browse recent logbook entries
- Real-time state updates
- French and English interface

## Quick start

1. Install the add-on from the repository
2. Start the add-on
3. Connect a Minitel emulator to `ws://<your-ha-ip>:3615/`

See [DOCS.md](DOCS.md) for full documentation.
